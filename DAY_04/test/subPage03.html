경제 뉴스 기사 분류

용도
사용자가 입력한 뉴스 기사의 내용을 분석하여
글로벌 경제, 금융, 부동산, 증권 중 해당하는 카테고리로 분류하는 시스템을 구축.



구성요소
이 프로젝트는 경제 관련 뉴스 기사 분류모델을 구현하기 위해 딥러딩 모델을 웹 환경으로 활용해
사용자 입력을 시시간으로 처리하고 결과를 제공하는 기능을 넣어 만들었습니다.

1. 목표설정
2. 데이터 전처리 및 토크나이저 설정 3. 모델 학습 준비
4. 웹 서버 및 CGI 설정

목표 설정

사용자가 입력한 뉴스 기사를 분석하여 해당 카테고리로 자동 분류하고,
그 결과를 웹페이지에서 실시간으로 제공하는 시스템을 구축하기.

데이터 전처리 및 토크나이저 설정

사용자가 입력한 뉴스 기사를 모델이 처리할 수 있는 형식으로 변환하기위해
transformers 라이브러리의 ElectraTokenizer를 사용했습니다.
분류된 결과를 사용자가 이해 할 수 있는 형태로 출력하기 위해 4개의 카테고리를 정의해주었습니다.

모델 학습

경제 관련 뉴스기사를 분류하기 위해 BERT+LSTM 모델을 사용했습니다.
BERT는 문장의 문맥을 반영한 텍스트 임베딩을 생성하는 사전 학습된 모델입니다.
LSTM은 문장의 순서과 관계를 반영하는 순환 신경망(RNN) 계열의 모델로
BERT의 출력값을더 세밀하게 학습할 수 있도록 사용했습니다.

웹 서버 및 CGI 설정

학습된 경제뉴스기사 분류 모델을 CGI 스크립트 내에서 로드됩니다.
모델은 학습결과가 저장된 파일을 불러옵니다.
입력된 문장을 전처리 및 토큰화 한 후 모델에 전달하여 뉴스기사를 분류합니다.
예측된 감정 분석 결과는 HTML 형식으로 웹 브라우저에 출력됩니다.
