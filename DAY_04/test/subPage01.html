트위터 댓글 감정분석 프로젝트

용도
트위터 댓글들의 감정들을 분석해서 긍정적인 댓글인지 부정적인 댓글인지 분류하는 모델입니다.
구성요소
프로젝트는 크게 세 부분으로 나눌 수 있습니다.

1. 웹 인터페이스 (HTML 페이지)
2. CGI 스크립트 (Python 기반의 서버 측 처리)
3. 기계 학습 모델 (PyTorch 기반 감정 분석 모델)

웹 인터페이스

사용자는 문장을 텍스트 입력 창에 입력하고,
분석 버튼을 클릭하면 데이터가 서버로 전송됩니다.

CGI 스크립트

CGI 스크립트는 서버에서 실행되는 Python 코드로
웹 브라우저로부터 데이터를 받아 처리하고 그 결과를 HTML로 반환합니다.
즉, 사용자가 데이터를 입력하면 전송된 텍스트를 받아와서
분석을 위해 불용어 제거, 소문자 변환, 특수문자 제거 등을 통해 전처리를 하는 구조입니다.

모델을 통한 감정 분석

PyTorch로 학습된 감정 분석 모델이 CGI 스크립트 내에서 로드됩니다.
모델은 학습결과가 저장된 파일을 불러옵니다.
입력된 문장을 TF-IDF 벡터화 한후, 모델에 전달하여 긍정 또는 부정으로 예측합니다.
예측된 감정 분석 결과는 HTML 형식으로 웹 브라우저에 출력됩니다.
