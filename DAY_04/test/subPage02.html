교통 표지판 분류 모델

용도
교통 표지판 분류를 위한 딥러닝 기반의 이미지 분류 모델을 구축하고
이를 웹 환경에서 사용자가 이미지를 업로드하여 예측 결과를 확인할 수 있도록 구현한것입니다.
구성요소
프로젝트의 흐름을 설명드리겠습니다.

1. 데이터 수집 및 전처리
2. 모델 설계 및 학습
3. 모델 평가 및 웹 인터페이스 구현

테이터 수집 및 전처리

Kaggle에 있는 교통표지판 이미지를 불러와서 데이터를 딥러닝 모델에 입력할 수 있도록
이미지에 대한 정규화와 텐서화를 진행했습니다.
전처리 단계에서 이미지 데이터에 대해 랜덤 회전, 수평 뒤집기등의 데이터 증강기법을 적용했습니다.

모델 설계 및 학습

CNN 모델을 사용하여 전처리한 이미지를 학습하게 했습니다.
2D 합성곱 계층으로 이미지의 특징을 추출해서
Bacth Normalization으로 학습 안정성을 높였구요.
Dropout을 설정해 과적합을 방지했고 Adam Optimizer를 사용해
모델 파라미터를 업데이트 하는 형식으로 모델을 구성했습니다.


모델 평가 및 웹 인터페이스 구현

학습된 모델을 사용해 정확도(Accuracy)를 평가한 후 정확도가 충분히 나왔을 때 모델을 저장했습니다.
CGI를 사용하여 웹 서버에서 교통 표지판 이미지를 업로드하고 분류하는 기능을 구현했습니다.
사용자가 이미지를 업로드하면 서버로 전송되고 모델이 이미지를 처리하여 예측 결과를 봔환합니다.
예측된 교통 표지판의 라벨과 업로드한 이미지를 웹페이지에서 확인 할 수 있습니다.
